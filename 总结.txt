git配置
------------------------
	git config             #简要帮助说明
	git config --help      #网页版详细帮助说明

	
创建版本库，把某个目录让git管理
----------------------------------------
	git init   
	
.git目录介绍
---------------------------------------
	
	.git/HEAD       本地仓库 当前引用
	.git/ORIG_HEAD  远程仓库 当前引用


基本使用
--------------------------------------
	#将工作空间的文件添加到缓存区
	git add file
	#将缓存区的文件提交到本地仓库	
	git commit file -m "注释"       
	git add -a            #将工作空间的文件添加到缓存区
	git commit -a         #将缓存区的文件提交到本地仓库
